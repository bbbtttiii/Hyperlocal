# Hyperlocal
A CRUD/MVC app that lets amateur meteorologists track their station observations.

## Getting Started

- Clone
- Run `bundle install`
- Run `rake db:migrate`
- Run `shotgun`

## Features

After creating an account, the user can make a new report as well as view their previous observations.
